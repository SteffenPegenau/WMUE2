    FAQ

    • Was kostet der E-Mail-Account von Unitymedia?

      Das E-Mail-Konto von Unitymedia gehört zu Ihrem Unitymedia Internetanschluss und es fallen dafür keine Zusatzkosten an.

    • Erhalte ich bei Unitymedia mehrere E-Mail-Postfächer?

      Insgesamt ist bei jedem Unitymedia Internetanschluss ein E-Mail-Account mit einem Postfach inklusive. Für dieses E-Mail-Postfach können bis zu fünf Alias-Namen (Weiterleitungsadressen ohne eigenes E-Mail-Postfach) angelegt werden.

    • Was passiert mit meiner E-Mail-Adresse, wenn ich meinen Internetanschluss bei Unitymedia kündige?

      Sollten Sie Ihren Internetanschluss bei uns kündigen, werden das E-Mail-Postfach und die E-Mail-Adresse unwiderruflich gelöscht. Aus datenschutzrechtlichen Gründen bleibt die E-Mail-Adresse für sechs Monate gesperrt und steht erst danach wieder für andere Interessenten zur Verfügung.

    • Wie starte ich die Guided Tour?

      Wo finde ich eine Anleitung zu Unitymedia Mail?

      Sie finden eine Anleitung nach der Anmeldung in den Einstellungen. Klicken Sie oben rechts auf folgendes Icon und wählen Sie „Guided Tour für diese App“ aus.

    • E-Mail-Passwort zurücksetzen

      Sie erinnern sich nicht mehr an Ihr Unitymedia Mail Passwort?

      Melden Sie sich dafür im Unitymedia E-Mail-Servicecenter an. Wählen Sie nun im obigen Menü "Email Konfiguration" aus. Danach suchen Sie unten in der Tabelle die E-Mail-Adresse aus, deren Passwort sie zurücksetzen möchten. Klicken Sie dann rechts auf den Link „Email-Account konfigurieren“.

      Auf der folgenden Seite klicken Sie auf den Link „Email Passwort ändern“. Es wird eine neue Seite geladen, in der Sie das neue Passwort eingeben können. Bestätigen Sie das Passwort im zweiten Eingabefeld und klicken Sie auf den Button „Passwort ändern“. Danach ist das Passwort für diese E-Mail-Adresse geändert und Sie können es für Unitymedia Mail verwenden. Wenn Sie ein externes E-Mail-Programm verwenden, ändern Sie das Passwort dort ebenfalls.

      Sie erinnern sich nicht an Ihr Unitymedia E-Mail-Servicecenter Passwort?

      Falls Sie sich nicht mehr an Ihr Unitymedia E-Mail-Servicecenter Passwort erinnern können, klicken Sie auf den Link „Beantworten sie Ihre Geheimfrage“ unter dem „Anmelden“ Button und folgen Sie den Anweisungen.

    • E-Mail-Adresse umbenennen / Wie richte ich einen Alias ein?

      Sie möchten den Namen Ihrer E-Mail-Adresse ändern?

      Melden Sie sich dafür einfach im Unitymedia E-Mail-Servicecenter an. Wählen Sie im Menü "Email Konfiguration" aus und klicken Sie anschließend auf den Link „Email-Account konfigurieren“ . Auf der folgenden Seite klicken Sie auf den Link „Alias hinzufügen“, um einen neuen Alias hinzuzufügen. Geben Sie dazu die Alias-Adresse in das Textfeld „Alias-Adresse“ ein und prüfen Sie die Verfügbarkeit mit einem Klick auf den Button „Verfügbarkeit prüfen“. Nach erfolgreicher Prüfung klicken Sie auf den Button „Alias hinzufügen“.

      Auf der gleichen Seite können Sie den Alias auch löschen. Klicken Sie dazu auf den Link „Alias löschen“ und bestätigen Sie auf der nächsten Seite, indem Sie auf den Button „Alias löschen" klicken.

    • Was ist der Unterschied zwischen einer normalen E-Mail-Adresse und einem Alias?

      Mit einem E-Mail Alias haben Sie die Möglichkeit, einer Ihrer bestehenden Unitymedia Mail E-Mail-Adressen bis zu fünf weitere Namen zu geben. Jeder dieser Namen ist ein Alias.

      Der Unterschied ist, dass ein Alias kein eigenes Postfach hat, er nutzt das Postfach der übergeordneten E-Mail-Adresse und somit landen Nachrichten, die an einen Alias geschickt werden auch in diesem Postfach. Der Alias hat somit auch keinen eigenen Usernamen bzw. Passwort, sondern funktioniert mit den Anmeldedaten der übergeordneten E-Mail-Adresse.

    • Weitere E-Mail-Adressen anlegen oder löschen

      Sie möchten sich neben Ihrer Hauptemailadresse gerne weitere E-Mail-Adressen anlegen?

      Melden Sie sich im Unitymedia E-Mail-Servicecenter an. Wählen Sie nun im obigen Menü „Email Konfiguration“ aus. Klicken Sie auf den Link „Email hinzufügen“ unten im Text, um eine neue E-Mail-Adresse hinzuzufügen. Nach der Eingabe der Adresse, klicken Sie dann auf den Button „Verfügbarkeit prüfen“. Nach erfolgreicher Prüfung werden Sie aufgefordert ein Passwort für diese E-Mail-Adresse einzugeben. Klicken Sie nach der Passworteingabe auf den Button „Email hinzufügen“. Bitte achten Sie dabei auf die unten aufgeführten Passwortregeln.

      Sie möchten eine E-Mail-Adresse löschen?

      Falls Sie eine E-Mail-Adresse löschen möchten, klicken Sie auf den Link „Email-Account konfigurieren“. Auf der folgenden Seite klicken Sie auf den Link „Email löschen“. Bestätigen Sie danach, indem Sie auf den Button „Email löschen“ klicken.

      Hinweis: Wenn Sie eine bestehende E-Mail-Adresse löschen, werden auch die damit verbundenen E-Mails gelöscht.

      Unitymedia Tipp: Ihre neu angelegten E-Mail-Adressen können Sie sich anschließend auch in Ihrem E-Mail Client (Outlook, Thunderbird, usw.) einrichten. Lesen Sie dazu den Punkt Konfiguration von einem externen Mail-Programm für Unitymedia Mail. In unseren E-Mail Einrichtungshilfen haben wir Ihnen Anleitungen für die gängigsten E-Mail-Programme bereitgestellt (Link).

    • Wie erhalte ich die Zugangsdaten?

      Bei der Bestellung eines Unitymedia Internetprodukts erhalten Sie automatisch Ihre persönliche Unitymedia E-Mail-Adresse. Die Zugangsdaten schicken wir an Ihre Postadresse.
